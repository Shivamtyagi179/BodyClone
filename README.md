🧠 Body Clone using OpenCV & Python

A real-time computer vision system that mirrors human actions using a webcam, creating a "body clone" effect.

🚀 Overview

The Body Clone Project is a real-time motion tracking system built using Python and OpenCV.
It captures human body movements via a webcam and replicates them instantly, creating a clone-like visual effect.

This project demonstrates the power of computer vision in understanding and mimicking human actions.

✨ Features
🎥 Real-time webcam input

🕺 Motion detection & tracking

🔁 Action mirroring (clone effect

⚡ Low-latency performance

🧩 Simple and modular code structure


🛠️ Tech Stack
Language: Python 🐍
Library: OpenCV
MediaPipe

⚙️ Installation

# Clone the repository
git clone https://github.com/shivam179/body-clone.git

# Navigate to project folder
cd body-clone

# Install dependencies
pip install -r requirements.txt

▶️ Usage
python main.py
Allow camera access 🎥
Perform movements in front of the camera
Watch the system replicate your actions in real-time
